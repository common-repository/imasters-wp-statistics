== Installation ==

How to install iMasters WP Statistics

e.g.

1. Upload `imasters-wp-statistics` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP Statistics' menu in WordPress
